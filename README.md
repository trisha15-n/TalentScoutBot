# TalentScoutBot
A professional, context-aware chatbot designed to streamline the initial recruitment process. This tool automates candidate information gathering and conducts a technical screening using Large Language Models.

KEY FEATURES :- 
1. Information Gathering - Collects the information about the candidate like Full Name, Email Address, Phone no., Tech Stack etc
2. Based on technical stack and experience using Llama-3.3-70b (via Groq API) generate role-specific technical questions.
3. Real time Progress Tracking - A dynamic sidebar updates as the candidate provides information.

TECHNICAL STACK :-
i) Language - Python
ii) Frontend - Streamlit
iii) LLM API - Groq
